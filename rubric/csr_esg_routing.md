# CSR / ESG Routing Rules

Combined badge per user preference. Format:

`CSR/ESG ✓ ([routing target])` — when applicable
`CSR/ESG —` — when not applicable

Use **generic routing targets** only — never named individuals.

---

## Capability → CSR/ESG routing target mapping

### CSR side (Foundation / community / employee)

| If capability is… | Routes to |
|---|---|
| Rural women's health, maternal/child health, screening (cervical, breast, diabetes) | **Foundation/Nand Ghar** or **HZL Sakhi** (Rajasthan-specific) |
| Skill development, vocational training, rural literacy | **Foundation** (Project Aarambh, Subhalaxmi) |
| Accessibility / disability assistive tech (sign language, hearing aids, vision) | **Group HR D&I** + Foundation |
| Plant-worker / employee mental health, MSK pain, neurodiagnostics, OHC services | **Group OHC** (Occupational Health Centres) |
| Local livelihood, community dev, agri (high-humidity regions only — VAL-J/BALCO Korba) | **Foundation** + BU CSR |
| Education tech for plant-adjacent communities | **Foundation/Nand Ghar** |
| Clean drinking water for rural communities | **Foundation** + Group Sustainability |
| Sports, recreation, talent development for community children | **BU CSR** (e.g., HZL Football Academy at Zawar) |

### ESG side (Sustainability / board agenda)

| If capability is… | Routes to |
|---|---|
| Decarbonisation: green hydrogen, RE storage, carbon capture, biomass | **Group Sustainability** (board ESG committee) |
| Water management, ZLD, recycling, net water positivity | **Group Sustainability** + BU |
| Air quality, emissions monitoring, dust suppression, fly ash utilisation | **Group Sustainability** + BU HSE |
| Biodiversity, mangrove, mine-reclamation, afforestation | **Group Sustainability** + Foundation |
| Tailings/TSF safety (post-Brumadinho/Samarco GISTM) | **Group Sustainability** (also Cross-BU) |
| Circular economy, waste-to-value (slag, red mud, fly ash) | **Group Sustainability** + R&D |
| Energy efficiency at plant level | **Group Sustainability** + BU |

## What does NOT get CSR/ESG

- Generic "this could help society" claims without concrete Vedanta programme/site/footprint match
- Capabilities that work for any company — must concretely match Vedanta's specific CSR programmes (Nand Ghar, Sakhi, OHC, Project Aarambh, Subhalaxmi) or active ESG board themes (decarb, water, GISTM, biodiversity)
- Routing only because capability is "low-tech" or "rural-relevant" without a real Vedanta hook

If you can't name a specific Vedanta programme or site footprint match, do NOT tag CSR/ESG ✓.

## Examples

**TanPrish (cervical cancer AI screening)** — `CSR/ESG ✓ (Foundation/Nand Ghar + HZL Sakhi)`
- Concrete match: Nand Ghar covers HZL Rajasthan, ESL Jharkhand, BALCO Chhattisgarh, VAL Odisha, IOB Karnataka rural women
- HZL Sakhi already does rural women's health outreach in Bhilwara/Rajsamand
- Specific programme overlap → ✓

**Sthyr Energy (Zinc-Air LDES)** — `CSR/ESG ✓ (Group Sustainability)`
- Decarbonisation + RE storage = direct ESG board theme
- Uses domestic Zn (HZL strategic alignment)
- Aligned with $5bn net-zero pledge → ✓

**Bubble Me (D2C magnesium wellness)** — `CSR/ESG —`
- Generic consumer wellness brand
- No Vedanta CSR programme hook, no ESG theme match
- "Could help health" is too vague → not a routing

**Krvvy (D2C inclusive innerwear)** — `CSR/ESG —`
- Consumer apparel; even D&I framing is a stretch
- No Vedanta programme footprint match
- Skip → no routing
